# Payroll System

A simple Python program that calculates **employee payroll** including gross pay, various deductions, and net pay, using **decision-making statements** and **loops**.  

The program supports **multiple employees**, processes payroll details interactively, and stops when the user enters **“done”**.

---

## Features
- Calculates **gross pay** with overtime (1.5× rate beyond 40 hours)
- Applies all **deductions**:
  - Income Tax (15%)
  - National Health Levy (2.5%)
  - District Tax (1%)
  - Educational Fund (₦0.5 per child above 3)
- Displays a detailed, formatted **payslip** for each employee
- Uses the **₦ (Naira)** symbol for currency
- Runs until user enters `"done"` or `"Done"`
